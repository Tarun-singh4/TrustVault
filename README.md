Trustvault

Reputation-Based Credit Engine

📌 Overview
CredSignal evaluates digital workers using GitHub activity to generate a Trust Score for creditworthiness.
It helps freelancers access loans without traditional credit history.

⚙️ Features
GitHub-based scoring
Trust Score calculation
Risk classification
Loan eligibility decision


🧠 Flow
GitHub Data → Scoring → Trust Score → Loan Decision


🏗️ Tech Stack

Node.js + Express
MongoDB
GitHub API
🚀 Run
npm install
node server.js
📡 API
POST /api/github/score
